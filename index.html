Arduino timelapse

Timelapse timer on Atmega8L

Download Download View Arduino timelapse on GitHub

Description

Timelapse timer can make foto with different intevals. It's based on ATmega8L 8MHz with special bootloader Arduino NG. This device has has 3 indicators that show interval length (1 sec - 1 hour).

Schematic

Board has small size and consists smd components. Supply is provided by 9V battery.
